# Set accessor for \`tbl_now\` class

Set accessor for \`tbl_now\` class

``` r

```

## Arguments

- x:

  A \`tbl_now\` object

- name:

  a literal character string or a
  [name](https://rdrr.io/r/base/name.html) (possibly
  [backtick](https://rdrr.io/r/base/Quotes.html) quoted). For
  extraction, this is normally (see under ‘Environments’) partially
  matched to the [`names`](https://rdrr.io/r/base/names.html) of the
  object.

- value:

  typically an array-like R object of a similar class as `x`.

## Value

A \`tbl_now\` object or a \`data.frame\`
